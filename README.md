# RPC Rust

## Requirements

Install protoc binaries

### MacOS
```bash
brew install protobuf
```

### Debian-based Linux
```bash
sudo apt-get install protobuf-compiler
```

### Install Just for commands
```bash
cargo install just
```

## Build

`cargo build`

## Run the integration example

`just run-integration`

## Run the multi language integration example

`just run-multilang`
